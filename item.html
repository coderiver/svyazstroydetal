Муфта МТОК-А1/216-1КТ3645-К-77 (проволочная броня)
Номенклатурный номер: 130103-00071

Описание

Компания «СВЯЗЬСТРОЙДЕТАЛЬ», основной российский производитель оптических муфт, продолжает производство магистральных оптических муфт МТОК-А1. Муфты представляют собой модернизированный вариант широко распространённой муфты МТОК 96-О1-IV.

В новой муфте используются те же детали корпуса, кронштейн, подкассетник и кабель-ные вводы №7 и №8. Но кассета изменилась. Теперь на ней не 32 места для КДЗС, а 36. Соответственно увеличилась до 216 сварок и общая ёмкость муфты.

Особенности:

  • муфта предназначена для сращивания подземных и подводных ОК с бронёй из стальных проволок;
  • может устанавливаться в колодцах, в котлованах и на дне водоёмов глубиной до 10 метров;
  • муфта имеет три патрубка для ввода ОК и три малых патрубка для вывода проводов заземления (КИП);
  • монтируется в вариантах «прямая муфта с вводом двух кабелей» и в варианте «разветвительная муфта с вводом трёх кабелей»;
  • корпус муфты и вводы кабелей герметизируются трубками ТУТ с подклеивающим слоем;
  • внутренние элементы муфты полностью соответствуют требованиям «Правил» 2006 г.;
  • муфты поставляются с двумя установленными кабельными вводами;
  • для сращивания двух подземных бронированных кабелей поставляется типоразмер 130103-00071. Муфта МТОК-А1/216-1КТ3645-К-77;
  • для сращивания подземного и подводного бронированных кабелей поставляется типоразмер 130103-00072. Муфта МТОК-А1/216-1КТ3645-К-78;
  • для сращивания двух подводных бронированных кабелей поставляется типоразмер 130103-00073. Муфта МТОК-А1/216-1КТ3645-К-88;
  • в базовом комплекте муфты одна кассета с гильзами КДЗС и два кабельных ввода;
  • для ввода в муфты третьего кабеля отдельно поставляются кабельные вводы №7 и №8;
  • для увеличения ёмкости муфты отдельно поставляются «Комплекты кассеты КТ3645» с гильзами, стяжками и маркерами;
  • для вывода из муфт проводов отдельно поставляется «Комплект провода заземления»;
  • для оборудования подземного скрытого КИП поставляется «Контейнер проводов заземления»;
  • для защиты оптической муфты при укладке её на дно водоёма отдельно поставляется муфта защитная МЧЗ и стальные крепления – кронштейн и фиксатор кабелей;
  • для защиты оптической муфты при установке её в котлованах отдельно поставляются защитные муфты МЧЗ или МПЗ (выбираются по типу грунта);
  • во всех вариантах установки защитные муфты заливаются вспенивающимся герметиком;
  • для установки муфты в новых колодцах отдельно поставляется «Кронштейн для установки тупиковых муфт в колодцах»;
  • для установки муфт в заполненных колодцах, под потолком, поставляются консоли типа КСО с крепежом;
  • для установки кабельных вводов отдельно поставляется «Ключ для монтажа МТОК;
  • для организации рабочего места для монтажа отдельно поставляются кронштейны и струбцины;
  • имеются инструкции по монтажу, как самой оптической муфты, так и кассеты типа КТ3645 и муфт защитных.
